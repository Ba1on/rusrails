h1. Настройка среды Rails

<p>Некоторые части Rails также могут быть сконфигурированы извне, предоставив переменные среды. Следующие переменные среды распознаются различными частыми Rails:</p>
<ul>
	<li><tt>ENV["RAILS_ENV"]</tt> определяет среду Rails (production, development, test и так далее), под которой будет запущен Rails.</li>
</ul>
<ul>
	<li><tt>ENV["RAILS_RELATIVE_URL_ROOT"]</tt> используется кодом роутинга для распознания URL при развертывании вашего приложение в поддиректории.</li>
</ul>
<ul>
	<li><tt>ENV["RAILS_ASSET_ID"]</tt> переопределит кэшевые временные метки по умолчание, которые Rails создает для скачиваемых ресурсов.</li>
</ul>
<ul>
	<li><tt>ENV["RAILS_CACHE_ID"]</tt> и <tt>ENV["RAILS_APP_VERSION"]</tt> используются для создания рсширенных ключей кэша в коде кэширования Rails. Это позволит иметь несколько отдельных кэшей в одном и том же приложении.</li>
</ul>
