h1. Расположение инициализационного кода

<p>Rails предлагает четыре стандартных места для размещения инициализационного кода:</p>
<ul>
	<li>config/application.rb</li>
	<li>Конфигурационные файлы конкретных сред</li>
	<li>Инициализаторы</li>
	<li>Пост-инициализаторы</li>
</ul>

<h3>Запуск кода до Rails</h3>
<p>В тех редких случаях, когда вашему приложению необходимо запустить некоторый код до того, как сам Rails загрузится, поместите его до вызова <tt>require 'rails/all'</tt> в <tt>config/application.rb</tt>.</p>
