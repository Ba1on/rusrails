h1. Навязывание протокола HTTPS

<p>Иногда хочется навязать определенному контроллеру быть доступным только через протокол HTTPS по причине безопасности. Начиная с Rails 3.1 можно использовать в контроллере метод <tt>force_ssl</tt>, для принуждения к нему:</p>
<div class="code_container"><pre class="brush: ruby; gutter: false; toolbar: false">
class DinnerController
  force_ssl
end
</pre></div>
<p>подобно фильтру, можно также передать <tt>:only</tt> и <tt>:except</tt> для принуждения безопасного соединения только определенным экшнам.</p>
<div class="code_container"><pre class="brush: ruby; gutter: false; toolbar: false">
class DinnerController
  force_ssl :only => :cheeseburger
  # или
  force_ssl :except => :cheeseburger
end
</pre></div>
<p>Пожалуйста, отметьте, что если вы добавили <tt>force_ssl</tt> во многие контроллеры, то, возможно, вместо этого хотите навязать всему приложению использование HTTPS. В этом случае можно установить <tt>config.force_ssl</tt> в файле окружения.</p>
