h1. Куки

<p>Ваше приложение может хранить небольшое количество данных у клиента &#8212; в так называемых куки &#8212; которое будет сохранено между запросами и даже сессиями. Rails обеспечивает простой доступ к куки посредством метода <tt>cookies</tt>, который &#8212; очень похоже на <tt>session</tt> &#8212; работает как хэш:</p>
<div class="code_container"><pre class="brush: ruby; gutter: false; toolbar: false">
class CommentsController &lt; ApplicationController
  def new
    # Автозаполнение имени комментатора, если оно хранится в куки.
    @comment = Comment.new(:name =&gt; cookies[:commenter_name])
  end

  def create
    @comment = Comment.new(params[:comment])
    if @comment.save
      flash[:notice] = &quot;Thanks for your comment!&quot;
      if params[:remember_name]
        # Запоминаем имя комментатора.
        cookies[:commenter_name] = @comment.name
      else
        # Удаляем из куки имя комментатора, если оно есть.
        cookies.delete(:commenter_name)
      end
      redirect_to @comment.article
    else
      render :action =&gt; &quot;new&quot;
    end
  end
end
</pre></div>
<p>Отметьте, что если для удаления сессии устанавливался ключ в <tt>nil</tt>, то для удаления значения куки следует использовать <tt>cookies.delete(:key)</tt>.</p>
