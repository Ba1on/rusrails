h1. Тестирование производительности приложений Rails

<p>Это руководство раскрывает различные способы тестирования производительности приложения на Ruby on Rails. Обратившись к нему, вы сможете:</p>
<ul>
	<li>Понимать различные типы метрик бенчмаркинга и профилирования</li>
	<li>Создавать тесты производительности и бенчмаркинга</li>
	<li>Использовать двоичный файл Ruby, пропатченого GC, для измерения использования памяти и размещения объектов</li>
	<li>Понимать бенчмаркинговую информацию, предоставленную Rails в файлах лога</li>
	<li>Изучить различные инструменты, помогающие бенчмаркигу и профилированию</li>
</ul>
<p>Тестирование производительности это неотъемлемая часть цикла разработки. Очень важно, чтобы конечные пользователи не ждали долго полной загрузки страницы. Обеспечение приятного серфинга для конечных пользователей и снижение расходов на ненужное оборудование важны для любого нетривиального веб приложения.</p>
