h1. Как разделить большой файл с маршрутами (Rails4)

Если имеется большой файл с маршрутами, который бы вы хотели разделить на несколько файлов, можно использовать метод +#draw+ в вашем маршрутном файле:

<ruby>
draw :admin
</ruby>

Затем создайте файл +config/routes/admin.rb+. Имя файла -- то же самое, что и символ, переданный в метод +draw+. Затем в этом файле можно использовать обычный маршрутный DSL:

<ruby>
# в config/routes/admin.rb

namespace :admin do
  resources :posts
end
</ruby>
