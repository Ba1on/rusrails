h1. Rails для начинающих

<p>Это руководство раскрывает установку и запуск Ruby on Rails. После его прочтения, Вы будете ознакомлены:</p>
<ul>
<li>С установкой Rails, созданием нового приложения на Rails и присоединением Вашего приложения к базе данных</li>
<li>С общей структурой приложения на Rails</li>
<li>С основными принципами <span class="caps">MVC</span> (Model, View Controller - «Модель-представление-контроллер») и дизайна, основанного на RESTful</li>
<li>С тем, как быстро создать изначальный код приложения на Rails.</li>
</ul>
<div class='warning'><p>Это руководство основывается на Rails 3.1. Часть кода, показанного здесь, не будет работать для более ранних версий Rails. Руководства для начинающих, основанные на Rails 3.0 и 2.3 вы можете просмотреть <a href="http://archive.rusrails.ru">в архиве</a></div>
